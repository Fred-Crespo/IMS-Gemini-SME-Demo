# Intelligent Management System (IMS) for SMEs

This repository contains the source code artifacts for the paper:
**"Digital Transformation in Telecommunications SMEs through Intelligent Management Systems"** published in IEEE Latin America Transactions.

## Contents
* **/src/Services:** Contains the PHP logic for connecting to Google Gemini API.
* **/docker:** Configuration files for reproducible deployment.
* **System_Prompt.txt:** The specific prompt engineering used for ticket classification.

## Note
This is a stripped-down version of the system intended for scientific reproducibility. Proprietary business logic and sensitive data have been removed.
